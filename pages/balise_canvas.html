<canvas>

Description Exemple Résultat de l'exemple

La balise <canvas>permet de dessiner des
éléments graphiques via un script (habituellement
en JavaScript ou grâce à WebGL). Il peut par exemple
être utilisé afin de dessiner des graphiques, manipuler des images
ou jouer des animations.

Le résultat de cette balise ne peut pas être visualisé.Elle est consommée par le navigateur web et il n'y a pas de rendu visible
