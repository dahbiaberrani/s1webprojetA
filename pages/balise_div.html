<div>

Description Exemple Résultat de l'exemple

La balise <div>(qui signifie division du document) est un conteneur générique qui permet d'organiser le contenu sans représenter rien de particulier. Il peut être utilisé afin de grouper d'autres éléments pour leur appliquer un style (en utilisant les attributsclass ou id).Il doit uniquement être utilisé lorsqu'aucun autre élément sémantique (par exemple<article> ou <nav>) n'est approprié.
