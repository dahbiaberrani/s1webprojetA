Description Exemple Résultat de l'exemple

La balise <ol> est employée pour créer une
liste numérotée.
La liste peut être numérique ou alphabétique.
La balise <ol> Balise de fermeture obligatoire
La balise <ol> est une liste ordonnée
d'une page web html (balise de type BLOC).
Même si la balise <ol> est une balise de type BLOC,
elle ne peut contenir que des balises <li>
